## Installation

1. Install [Tampermonkey](https://tampermonkey.org/) for your browser
2. Open the script file and copy the contents
3. Create a new userscript in Tampermonkey and paste
4. Save and enable

## Supported Domains

- `stake.com` and all subdomains
- `stake.games`
- `stake.ac`
- `stake.mba`
- `stake.pet`
- `stake.bet`

## Features

- Currency display formatting
- Mobile + PC compatible
- Auto-detects active currency
